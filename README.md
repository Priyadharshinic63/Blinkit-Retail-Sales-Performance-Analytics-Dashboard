# 📊 Blinkit Retail Sales & Performance Analytics Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard developed to analyze Blinkit's retail sales performance. The dashboard provides meaningful insights into sales, customer ratings, outlet performance, and product categories through dynamic visualizations. It helps users monitor business performance and supports data-driven decision-making.

## 🚀 Key Features
- Interactive dashboard with slicers and filters
- Sales performance analysis across different outlet locations
- Product category and item type analysis
- KPI cards for Total Sales, Average Sales, Total Items, and Average Rating
- Outlet establishment trend analysis
- Easy-to-understand business insights using charts and graphs

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Dashboard Insights
- Analyze overall sales performance
- Compare outlet performance by size and location
- Identify top-performing product categories
- Monitor customer ratings and average sales
- Visualize business trends for better decision-making

## 🎯 Project Outcome
This dashboard transforms raw retail sales data into meaningful business insights using interactive visualizations. It enables users to track key performance metrics, identify trends, and make informed business decisions efficiently.
